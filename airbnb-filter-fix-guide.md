# 🚨 AKUT GUIDE: FIXA AIRBNB-FILTER I GMAIL

## **STEG 1: KONTROLLERA EXISTERANDE FILTER**

1. **Öppna Gmail** i webbläsaren
2. Klicka på **kugghjulet (⚙️)** uppe till höger
3. Välj **"Se alla inställningar"**
4. Gå till fliken **"Filter och blockerade adresser"**
5. **Leta efter Airbnb-filter** - titta på:
   - Filter som innehåller "airbnb" i namnet
   - Filter med villkor som `*@airbnb.com`
   - Filter som flyttar mail till papperskorgen eller arkivet

**📝 Rapportera här vad du ser:**
- Finns det några Airbnb-filter?
- Vilka villkor har de?
- Vilka åtgärder utför de?
- Är de aktiverade eller inaktiverade?

## **STEG 2: SKAPA NYTT, ENKLARE FILTER**

**Om INGET filter finns eller om det befintliga inte fungerar:**

1. Klicka på **"Skapa ett nytt filter"** längst ner på sidan
2. I fältet **"Från:"** skriv: `*@airbnb.com`
3. Klicka på **"Skapa filter"**
4. Markera **"Flytta till papperskorgen"**
5. Klicka på **"Skapa filter"** igen

## **STEG 3: TESTA MED EXISTERANDE MAIL**

1. Gå tillbaka till **inkorgen**
2. **Välj ett Airbnb-mail** (markera kryssrutan)
3. Klicka på **"..."** (tre punkter) ovanför maillistan
4. Välj **"Filter liknande mail"**
5. Ett nytt filterfönster öppnas med fälten ifyllda
6. Kontrollera att **"Från:"** fältet innehåller `*@airbnb.com` eller liknande
7. Markera **"Flytta till papperskorgen"**
8. Klicka på **"Skapa filter"**

## **STEG 4: ALTERNATIVA FILTERREGELER**

**Skapa MULTIPLA filter för att fånga alla varianter:**

1. **Filter 1:** `*@airbnb.com` → Flytta till papperskorgen
2. **Filter 2:** `*@airbnb.se` → Flytta till papperskorgen  
3. **Filter 3:** `*@airbnb.co.uk` → Flytta till papperskorgen
4. **Filter 4:** `*@airbnb.fr` → Flytta till papperskorgen
5. **Filter 5:** **"Airbnb" i ämnesraden** → Flytta till papperskorgen

**Så här skapar du filter 5:**
- Klicka "Skapa ett nytt filter"
- I fältet **"Har orden:"** skriv: `Airbnb`
- Klicka "Skapa filter" → "Flytta till papperskorgen" → "Skapa filter"

## **STEG 5: MANUELL ÅTGÄRD OMEDELBART**

**Medan filter konfigureras:**

1. I inkorgen: **Markera alla Airbnb-mail**
   - Använd sök: `from:airbnb OR subject:airbnb`
   - Klicka på kryssrutan högst upp (markerar alla på sidan)
   - Klicka "Markera alla konversationer som matchar sökningen"

2. Klicka på **papperskorgsikonen** (🗑️) för att flytta till papperskorgen

3. **Från papperskorgen:**
   - Öppna papperskorgen
   - Markera ett Airbnb-mail
   - Klicka "..." → "Filter liknande mail"
   - Skapa filter som flyttar framtida mail direkt till papperskorgen

## **STEG 6: TESTA FILTERN**

1. **Vänta på nytt Airbnb-mail** eller be någon skicka testmail
2. **Kontrollera att det hamnar i papperskorgen** direkt
3. Om inte: gå tillbaka till STEG 1 och kontrollera filterinställningarna

## **VANLIGA PROBLEM OCH LÖSNINGAR**

### ❌ **Filter skapas men fungerar inte**
- Kontrollera att filter är **aktiverade** (grön knapp)
- Testa med **"Filtrera nu"**-knappen på filterinställningssidan
- Kolla om det finns **konflikter med andra filter**

### ❌ **Mail hamnar i "Skräppost" istället**
- Gå till "Skräppost"-mappen
- Markera ett Airbnb-mail
- Klicka "Inte skräppost"
- Skapa filter som **överstiger** skräppost-filter

### ❌ **Filter matchar inte alla Airbnb-mail**
- Lägg till fler domäner: `*@airbnb.*` (fångar alla länder)
- Lägg till specifika avsändare du sett
- Använd **"Från:"** OCH **"Ämne:"** för bredare matchning
